# IG-OSINT-V2

## INSTALLATION COMMANDS
```
apt update && apt upgrade -y
```
```
pkg install git python -y
```
```
termux-setup-storage
```
```
git clone https://github.com/Achik-Ahmed/IG-OSINT-V2
```
```
cd IG-OSINT-V2
```
```
pip install instaloader
```
```
python ig.py
```
